About Me

My name is Rasul, I'm currently a Software Engineer at Guy Carpenter where I help with

  • designing and deploying modern data lake and related tooling on Azure
  • developing Django/React applications with K8S deployment
  • modeling datasets for easy consumption by the rest of the organization via BI tools

In my free time, I love to do more web dev. More specifically, I like to develop small and fun projects that help me learn new technologies. You can see a list of my projects here.

As a programmer, I learned to love writing. I try to write regularly, be it tech tutorials or about any other topic. An extension of written tech tutorials was to start making videos. For that, you can check my Youtube channel.

I like to experiment a lot. Trying something new is a passion of mine. Some of my recent successes were with brewing Kombucha, fermenting different foods, and baking bread. On the failure side of things, I can't keep a plant alive for more than a week.
